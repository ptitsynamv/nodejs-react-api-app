# Nodejs React API app

NodeJS v20.11.0, NPM v10.2.4

## Developing

### Backend

1. `cd .\backend`
2. `npm i`
3. `npm start`. Open http://localhost:8080

### Frontend

1. `cd .\frontend`
2. `npm i`
3. `npm start`. Open http://localhost:3000

### Server

1. `cd .\server`
2. `npm i`
3. `npm start`. Open http://localhost:3001
